Este programa es un editor y analizador de código fuente diseñado para el lenguaje de programación RALL, como parte de un proyecto escolar para la materia de Lenguajes y autómatas II de la carrera de Ingeniería en Sistemas Computacionales del Instituto Tecnológico de Veracruz.
